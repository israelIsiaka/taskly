# Taskly

A macOS task management application.

## Installation

To build the app and install it:

```bash
./create_dmg.sh
```

This script will:
- Build the app for Universal (Apple Silicon + Intel)
- Create a DMG file
- Automatically install the app to `/Applications`
- Remove the quarantine attribute (requires sudo password)

## Requirements

- macOS 10.15 or later
- Xcode (for building from source)
