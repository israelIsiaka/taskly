# Utilities

This folder contains utility classes, extensions, and helpers.

## Current Files
- `ColorPalette.swift` - Color palette definitions and extensions
- `FontHelper.swift` - Sora font registration and extensions
- `SFSymbols.swift` - SF Symbols constants and helpers

## Future Utilities
- `GlassEffectModifier.swift` - Glass effect view modifier
- `AnimationHelpers.swift` - Animation utilities
- `DateExtensions.swift` - Date formatting helpers
- `ValidationHelpers.swift` - Input validation utilities
